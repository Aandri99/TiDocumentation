3.3.2. Network Manager

Note

We recommend using the Network Manager application to configure Red Pitaya connection settings. However, if local network access, which is needed to run the Network Manager application, is not available, it is still possible to connect manually.

Red Pitaya boards are network attachable devices focused on simple connectivity and quick accessibility. A graphical user interface for your Oscilloscope, Signal Generator, LCR meter, and other Red Pitaya applications directly on your PC without limitations such as limited commands or controls or any installation of additional software will provide you with a unique working experience.

You can connect to the Red Pitaya boards via:

  1. Local Area Network (LAN) - Requires a DHCP server on your LAN router

  2. Direct Ethernet cable connection - Requires additional settings on users’ PC and the Red Pitaya board

  3. Wireless Network Client - Requires an additional WiFi dongle available at the Red Pitaya store

  4. Access Point Mode (currently not supported) - The Red Pitaya board creates its WiFi network
