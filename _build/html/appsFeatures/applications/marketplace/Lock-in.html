2.2.12.1.5. Lock-in + PID (by Marcelo Luda)

Lock-in+PID is an application for the Red Pitaya STEMlab 125-14 board that implements an Oscilloscope application and a Lock-in amplifier. It’s based on relese 0.95 of the scope application.

More about this project can be found here:

Lock-in + PID

The Lock-in + PID application is available on the Red Pitaya marketplace.
