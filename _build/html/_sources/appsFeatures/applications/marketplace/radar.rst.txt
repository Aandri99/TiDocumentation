****************************
Open-source Red Pitaya Radar
****************************

Radar using Red Pitaya for RF and Raspberry Pi 3 / 4 for quad-core signal processing. Initially used for HF ionospheric imaging, but can be used at microwave and other frequencies via frequency translation.
