4.1.1.14. Known hardware issues

In this section is a list of known hardware issues with the Red Pitaya platforms. These issues will be fixed with the next hardware iteration of the boards.

4.1.1.14.1. Potential I2C system failures

Red Pitaya uses a TCA9406DCUR level translator between PS I2C pins and the I2C pins on the extension connector. TCA9406DCUR has a rise time accelerator built into it that is a non-standard feature of a level translator.

A combination of physical design (i.e. bus capacitance) and interaction between the two buffer’s rise time accelerators may cause I2C system failures. See the pink trace in the image below for the behavior-interaction effect due to 2 rise time accelerators on the bus.

4.1.1.14.2. UART TX preventing connection

If the UART TX pin on the E2 connector is driven high (3V3) before or during the boot sequence, this can prevent the user from logging into the unit.
